## メモ / 注意事項

- API Key・Secret は `.env` で管理し、**絶対に DB や Git に含めない**
- Hyperliquid Testnet で十分テストしてから本番に移行する
- LLM API のコストを監視し、不要なリクエストを最小化する
- ポジションサイズは必ず固定パーセントモデルで計算し、過大投入を防ぐ
